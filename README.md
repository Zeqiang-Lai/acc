# ACC

Another C Compiler Written in C++20

## Requirements

Windows

```shell
wget https://apt.llvm.org/llvm.sh
chmod +x llvm.sh
sudo ./llvm.sh 13

sudo apt-get install zlib1g zlib1g-dev
```
